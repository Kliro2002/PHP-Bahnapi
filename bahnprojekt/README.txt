# Projekt Bahnauskunft#


Funktion

Mit dem Programm können Sie auf die Schnittstelle der deutschen Bahn zugreifen.
Dabei gibt es die Möglichkeit einen Bahnhof auszuwählen, und genauere Informationen anzuzeigen.


Vorraussetzung
Das Programm wurde in PHP/HTML geschrieben und zur lokalen Verwendung geschrieben.
Serverseitige Verwendung wurde nicht geprüft.

Anleitung zum Start mit "XAMPP"

1. Installieren Sie XAMPP, falls noch nicht installiert.
2. Starten Sie den Apache Server
3. Legen Sie den Ordner "Bahnprojekt" in den htdocs-Ordner vom XAMPP
4. Rufen Sie "localhost/bahnprojekt/index.php" auf
